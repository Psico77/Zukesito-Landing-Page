Tecnologías dominadas

Java

Java es un lenguaje de programación orientado a objetos que se utiliza ampliamente para desarrollar aplicaciones de escritorio, móviles y web. Es conocido por su portabilidad y eficiencia, siendo la plataforma de desarrollo preferida para muchas empresas y desarrolladores.

C++

C++ es un lenguaje de programación de propósito general que se utiliza para crear aplicaciones de alto rendimiento. Fue desarrollado por Bjarne Stroustrup como una extensión del lenguaje C y es conocido por su capacidad de programación orientada a objetos.

Figma

Figma es una herramienta de diseño colaborativo basada en la web que permite a los usuarios crear interfaces y prototipos en tiempo real. Es utilizada ampliamente por equipos de diseño y desarrollo para mejorar la colaboración y la eficiencia en el proceso de diseño.

Tabletop creator

Tabletop Creator es un software diseñado para crear, prototipar y probar juegos de mesa, que permite a los usuarios diseñar componentes y exportarlos para juego físico u online. Ofrece herramientas para personalizar cada detalle de tu juego, facilitando dar vida a muchas ideas.

Godot

Godot es un motor de juegos de código abierto que permite a los desarrolladores crear juegos tanto en 2D como en 3D en diversas plataformas, incluyendo escritorio, móvil y web. Es gratuito bajo la Licencia MIT, lo que significa que no hay regalías ni tasas de licencia involucradas.

MCreator

MCreator es una herramienta de código abierto para crear mods de Minecraft Java Edition, complementos de Bedrock Edition y paquetes de datos utilizando una interfaz visual o un editor de código integrado.

HTML

HTML es el lenguaje estándar utilizado para estructurar y presentar contenido en la web. Define la estructura de las páginas web mediante elementos y etiquetas que permiten incluir texto, imágenes, videos y formularios.

CSS

CSS es un lenguaje de estilos utilizado para describir la presentación de documentos HTML o XML, permitiendo definir cómo se deben mostrar los elementos en pantalla, papel u otros medios.

JavaScript

JavaScript es un lenguaje de programación de alto nivel que se utiliza principalmente en el desarrollo web para crear experiencias interactivas en las páginas web. Es conocido por su capacidad para manipular el DOM y crear aplicaciones del lado del cliente.

MySQL

MySQL es un sistema de gestión de bases de datos relacional que se utiliza ampliamente en el desarrollo web para almacenar y gestionar información.

Bash

Bash es el intérprete de comandos por defecto en sistemas operativos Linux y macOS, utilizado para ejecutar comandos y scripts.

Claude AI

Claude AI es un modelo de lenguaje de gran tamaño desarrollado por Anthropic, diseñado para ser útil, seguro y confiable en una amplia variedad de tareas.

Pixel Studio

Pixel Studio es un software de edición de imágenes y diseño gráfico que permite crear y manipular imágenes en formato raster.

Tecnologías que quiero aprender

Github

Github es una plataforma de desarrollo de software que ofrece servicios de control de versiones y colaboración para proyectos de código abierto y privado.

Lua

Lua es un lenguaje de programación ligero, embeddable, que se utiliza principalmente para scripting en aplicaciones de juego y software.

Python

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general, ampliamente utilizado en el desarrollo web, ciencia de datos, inteligencia artificial y automatización.

Unity

Unity es un motor de videojuegos multiplataforma que permite crear experiencias interactivas para diversas plataformas.

Unreal Engine

Unreal Engine es un motor de videojuegos desarrollado por Epic Games que permite crear experiencias interactivas de alta calidad para diversas plataformas.
